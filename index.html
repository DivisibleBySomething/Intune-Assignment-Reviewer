Intune Assignment Dashboard

Sign in with your Microsoft 365 account to review Intune mobile app assignments across your tenant.

🔐
Sign in securely
Uses Microsoft's own login popup. No passwords or tokens are ever stored by this app.
📊
Reads from Microsoft Graph API
Intune app assignments are fetched directly from Microsoft's API using delegated permissions — the app only sees what you have access to.
🌐
Stays in your browser
All data is processed locally. Nothing is sent to any external server — this is a fully client-side app.
⚡
Session caching
Data is cached in your browser session for speed. Clicking Refresh or signing out clears it completely.
